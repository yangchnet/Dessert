croc: 跨网络、跨系统的空投

December 21, 2021 · 1 min · 李昌
Table of Contents
  • 1. install
  • 2. basic usage
  • 3. comment

1. install#

curl https://getcroc.schollz.com | bash

或

go install github.com/schollz/croc/v9@latest

2. basic usage#

sender:

$ croc send [file(s)-or-folder]
Sending 'file-or-folder' (X MB)
Code is: code-phrase

receiver:

croc code-phrase

3. comment#

可用于替代ftp上传文件，用于不同主机之间文件共享，类似“空投”的效果
