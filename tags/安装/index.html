安装docker-compose

安装docker-compose 下载最新版本的docker-compose sudo curl -L "https://github.com/docker/compose/releases/download/1.23.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose 对二进制文件应用可执行权限 sudo chmod +x /usr/local/bin/docker-compose 测试安装完成 docker-compose --version
